# 基于SpringBoot校园二手书管理系统
## 使用技术
由Maven构建的项目，技术栈是由SpringBoot+Mybatis+Thymeleaf组成，其中项目引入了一些中间件，如PageHelper（分页插件）、Lombok（数据层配置）、easyexcel（导出excel形式数据）

## 如何获取源码？
关注公众号《源码小客栈》，回复"校园"即可
